## üìù How to Capture Screenshots

### Method 1: Browser Screenshot Tools

1. Open the application in your browser
2. Press `F12` to open Developer Tools
3. Press `Ctrl+Shift+P` (Windows) or `Cmd+Shift+P` (Mac)
4. Type "screenshot" and select "Capture full size screenshot"

### Method 2: Windows Snipping Tool

1. Press `Windows + Shift + S`
2. Select the area to capture
3. Save the screenshot to this folder

### Method 3: Browser Extensions

- **Awesome Screenshot** - Chrome/Firefox extension
- **Fireshot** - Full page screenshots
- **Nimbus Screenshot** - Advanced capture tools
